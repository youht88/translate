请求的URL是 https://{域名}/ams/api/{版本}/{端点} ，其结构如下：

  • 域名 ：指由钱包后端分配的正式 域名 。
  • 版本 ：指API的 版本号 ，例如 v1 或 v2 。
  • 端点 ：指接口的 具体路径 ，例如 /{版本}/payments/pay 。

通过端点可以唯一识别一个接口。例如， /v1/payments/pay 与 /v2/payments/pay 是不同的。

  • 算法 ：指定用于生成签名的数字签名算法。 支持 RSA256 。
